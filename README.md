# Joke-generator-Vanilla-JS
A small exercise with API.  
## Features
Fetches random jokes from a public API
Uses Fetch API for asynchronous requests
Dynamically updates the DOM
Handles errors gracefully in the console

##How It Works
User clicks the button
A request is sent to the joke API
The response is converted to JSON
The joke (setup + punchline) is displayed on the page

##Technologies Used
HTML
CSS
JavaScript (Vanilla JS)
Fetch API

##Notes
Requires an internet connection (uses external API)
Errors are logged in the browser console

##Future Improvements
